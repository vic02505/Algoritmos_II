La nota de los TDAs es un valor numérico entre 0 y 2, siendo 0 el valor mínimo, y
2 el valor máximo.

Si se desea ejecutar algunos de los TDAs basta con ejecutar el comando "make" en la carpeta correspondiente a cada TDA. Vale añadir que lo que se hace al correr un TDA  es ejecutar las pruebas del mismo. Lo más importante del cada TDA viene en su correspondiente código, no en un uso del mismo para algún determinado fin. 

Para poder ver una aplicación de los TDAs se puede revisar la carpeta TPs de este repositorio.
